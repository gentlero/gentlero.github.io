Installation

Requirements

  • PHP >= 5.3 with following extensions enabled:
    • cURL
    • OpenSSL
  • Buzz library.
  • PHPUnit to run tests. (optional)

Composer

This library is available on Packagist and can be installed using Composer

  $ composer require "gentle/duckduckgo-api"
